# Intro
Breakfast Checkers is  checker game with a breakfast theme(there is no multiplayer or LAN currently, and yes im working on it).
there is 2 Team,cake(white) and burger(black)
team cake always move first

# How to start Game
1.Using Unity Play URL(https://play.unity.com/en/games/73c89e4a-231f-4c2c-a404-9ce0f4129f7d/breakfast-checkers).
2.from /build and run BreakfastChecker.exe.

# How to Play
1.There is An indicator(A Carrot) above the piece that determine which piece can move and which piece doesn't.
2.Drag Piece To the Highlighted Tile(green=Move,red=Attack) to Move.

# Rules
1.If you can Attack you must attack( you can only move with a piece that can attack).
2.If Piece reach opposing End of Board it will be Promoted(If piece reach end of board after attacked a piece and able to attack again as a king that piece may move again).
3.Game end when cake or burger have 0 piece or there is no legal move left.
